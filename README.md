# ML Mock Interview

Голосовой тренажёр собеседований для NLP/LLM инженера.

## Быстрый старт

1. Скопируй `config.example.js` → `config.js`
2. Вставь Gemini API ключ из [aistudio.google.com](https://aistudio.google.com)
3. Открой `index.html` в Chrome

## Структура

```
index.html          — приложение
questions.json      — 330 вопросов
config.example.js   — шаблон ключа
config.js           — твой ключ (в .gitignore)
```

## Режимы

- **Обычный** — 20-25 вопросов, глубокое доспрашивание
- **Жёсткий квиз** — 30-40 вопросов, быстрый темп
- **Быстрый** — 10-15 вопросов

## Темы

Python · ML · Deep Learning · Attention · Transformer · RAG · Fine-tuning · Agents · Alignment · Serving · Evals · System Design
